Utilicemos las funciones en R para obtener los cuartiles, para ello utilizaremos la funcin quantile que acepta 2 parmetros: vector de valores y la frecuencia relativa acumulada, as por ejemplo, si queremos el Q1 que es el valor que acumula el 25% de las observaciones se escribe: quantile(ingreos, 0.25)

Si deseamos obtener el Q2 que acumula el 50% se escribir: quantile(ingresos, 0.50)

Cmo se le conoce a Q2?

El Rango Intercuartlico, RIC, se define como la diferencia de Q3 Q1, calcule este valor.

Los bigotes se obtienen multiplicando el RIC * 1.5 y sumando a Q3 o restando a Q1, los valores que se encuentran fuera de valores se denominan atpicos.
